===============================
Archival description management
===============================

ESSArch offers a complete solutions for archival collections management. It adheres to ICA standards (ISAAR-CPF, ISAD-G, ISDF)
and has functionality for managing accessions, locations, classifications and preservation planning of archival holdings.
